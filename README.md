Some examples that show the use of different ML tools are included here.

## Machine Learning Glossary

For a quick refresher of ML terms, see this Google website: https://developers.google.com/machine-learning/glossary

## Supervised Learning

### Iris

See the Wikipedia page at https://en.wikipedia.org/wiki/Iris_flower_data_set for further info on the Iris data set. This data set is a good one to begin your machine learning journey with.

A jupyter-notebook is included in folder __Iris__.

### Medical

Nice classification example using a medical dataset. Jupyter-notebook and the `.csv` file is included in this folder.
